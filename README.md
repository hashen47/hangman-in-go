# Hangman (with golang)

## Description
- simple terminal base game with golang

## Screenshots
- ![Game Start](images/game_start.png)

## Dependencies
- golang 
- git (for clone the repo)

## Purpose
- learn more about golang basic syntax
- learn how to write testcases and run them

## Setup
- clone the repo `git clone https://github.com/hashen47/hangman-in-go.git`
- then to go the repo folder and run `go run .`
